Straddling Checkerboard Cipher

The Straddling Checkerboard Cipher is an awesome substitution cipher that turns letters into numbers then back into letters again.

The key for this cipher is a combination of a grid of letters and a secret key. The grid is created by writing the numbers 0-9 in the first row, followed by three rows of letters. Often people randomise the numbers in the first row. The letters are written in any order but lots of people write them in descending order of use (i.e. the most common letters first). The secret number can be any number, including numbers that start with zeros (e.g. 0452). In the first row of letters, skip two columns at random, label the second row of numbers with the number of the first column you skipped, label the second with the second. See below for an example grid.

To encrypt a message go through your plain text letter by letter. Find each letter in your grid, if the letter is in the first row of letters then write down the number that is above it in the grid. If the letter is in one of the other two letter rows then write down first the number label for its row then its column label. When you have gone through your entire plain text message you will have a long row of numbers. Take your secret number and write it repeatedly underneath these numbers. Now take each number one by one and add it to the number underneath it using modular arithmetic mod 10. Finally, take this new long line of numbers and use the grid to convert it into letters again.

To decrypt a message you do the opposite of what you did when encrypting. First, convert the letters into numbers, then take away the secret number (mod 10) and finally convert the numbers back into letters.
