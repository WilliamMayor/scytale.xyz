Playfair Cipher

The Playfair cipher is an interesting cipher that replaces single letters with a pair of letters. This makes hacking the encryption much harder than most substitution ciphers.

A Playfair cipher key is a 5x5 grid filled with letters. The grid only has space for 25 letters in total, so you can choose to either ignore the letter Q or to use I and J interchangably. To fill the grid, think of a phrase and write the phrase, letter by letter, into the grid. When you come to a letter that you have already written down, skip it. If you reach the end of your message and you still have spaces in your grid. Simply fill in the empty spaces with the left over letters in alphabetical order. You should end up with a grid like the one below.

To encrypt a message take the first two letters and apply the following rules:

  • If the letters are the same, add an X after the first one (e.g. AA would become AXA)
  • If the letters are on the same row of the key, replace them with the letters to their right. Wrap around the table if needed.
  • If the letter are in the same column, replace them with the letters below. Wrap around the table if needed.
  • If the letters are on different rows and columns then replace the letters with the letter that lies on the same row but it is in the column of the other.

Repeat this method for each pair of letter in the message. Add an extra X if you are left with a single letter.

To decrypt simply do the opposite of the last three rules above. You'll need to get rid of any odd looking Xs
