Affine Cipher

An Affine cipher is a very simple substitution cipher that uses a padding scheme and modular arithmetic.

The secret key for the Affine cipher is made up of three numbers; a, b, and c. When creating a key you can pick any number at all for b. There is a complicated relationship between a and c that restricts which numbers you can choose. To help you out, the key section below automatically calculates c when you give a value for a. If you type a value for a in the box below and no value for c is automatically produced, you need to choose a different a value.

To encrypt a single letter, first use the padding scheme to turn the letter into a number. Multiply that number by a and then add b. The resulting number could be bigger than 100, this would mean that the padding scheme cannot turn it into a letter. To avoid this, calculate your number modulo 100. Now you can safely use the padding scheme to produce a cipher letter. Repeat this for every letter in your plain text message and combine each of the cipher letters to form your cipher text.

To decrypt a single letter, you need to do the opposite of the encryption. Start by turning the letter into a number using the padding scheme. Now take away b then multiply by c. As you did before, calculate the result modulo 100 and turn that number into a letter using the padding scheme. Apply this decryption process to every letter in the cipher text to decrypt the whole message.
