Trifid Cipher

Trifid ciphers are one of the most complicated substitution ciphers in this list. They work by splitting each letter into 3 parts, each part then interacts with other parts from other letters to form a secure encryption.

The key for a Trifid cipher is a table with three, 3x3 boxes. Each cell in the table is filled up randomly with letters. Each letter can be found by using a 3-part coordinate; (which box, which row, which column).

To encrypt a message write out the letters in a line, underneath each letter write the letter's coordinates in a column. Now read off, from left to right, all of the coordinates. For each 3 numbers you read look up the letter that corresponds to that triplet

To decrypt convert each letter in the cipher text to a coordinate triplet and write the triplets out, from left to right, forming three equal-width rows. Now read down each column to retrieve the original triplet and use your table to convert it into the original letter.
