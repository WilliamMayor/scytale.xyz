Cryptanalysis - Mixed Alphabet Cipher

The tools on this page should help you break into a ciphertext encrypted using the Mixed Alphabet cipher.

In the box below put all of the ciphertexts that you have, one on each line.

Make sure that all of the ciphertexts have been encrypted using the Mixed Alphabet cipher.

Make sure that all of ciphertexts have been encrypted using the same key.

Here you can add the key to be used to decrypt the above ciphertexts. For parts of the key that you don't know yet, type a question mark (?).

The third line contains a randomly generated key that you can use to fill in the blanks. Use the "Generate Key" button to try a new random key.

Press this button to have the ciphertexts decrypted using one of the keys above.

Where a part of the key is unknown a question mark will be shown.
